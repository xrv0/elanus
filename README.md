## Installation

SSH into your Pi and execute the following commands.

```bash
git clone https://github.com/xrv0/elanus.git
cd elanus
pip3 install -r requirements.txt
```

## Usage

Run the following command to start the frontend aswell as the backend

```bash
python3 main.py
```
